02 · The insight

It's cheaper to check an answer
than to write one.

Writing a good answer is hard. But checking if an answer is right can be easy. With code, you can simply run it and see if it works. So we let a cheap AI take the first try, then check its work automatically. If it passes, we just got a great answer for almost nothing.

Code → run the tests Law → does the case it cited actually exist? Medicine → does it match trusted sources?
